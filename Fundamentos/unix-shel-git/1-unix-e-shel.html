Unix & Shel

Dividimos a introdução em cinco partes

Para começar, você vai estudar um famoso tipo de sistema operacional chamado Unix. Para isso, vamos colocar a mão na massa e executar alguns comandos no terminal. Depois de entender como os comandos funcionam e ter criado seus primeiros scripts, você vai aprender a usar o Git para guardá-los de forma segura. Em seguida, vai começar a parte mais divertida: o desenvolvimento web. Aqui, você vai estruturar documentos HTML, estilizá-los com o CSS e criar comportamentos customizados com o JavaScript. Após esse ciclo, você aprenderá sobre a criação de sites responsivos com CSS elegante e eficiente. Por fim, vai aprender a utilizar o JavaScript integralmente, incluindo HOF, código assíncrono, testes unitários e outros recursos avançados.

Você será capaz de:

Utilizar o Shell; Operar o Git e o GitHub confortavelmente; Criar documentos HTML estilizados com CSS; Usar o JavaScript para adicionar comportamento às páginas web; Utilizar recursos de HTML e CSS avançados; Trabalhar com JavaScript ES6; Escrever testes unitários; Manipular conjuntos de dados para resolver problemas complexos.

O que vamos aprender?

Nesta seção, você vai aprender: Conceito de Unix, Kernel e Shell; Quais os principais comandos do Unix; Como trabalhar no terminal; Como manipular diretórios e arquivos; Como exibir conteúdos de arquivos no terminal; Conceitos de Git e de GitHub; Como gerenciar versões de um projeto; Como criar repositório de projetos.

Terminal comandos

Principal ferramenta para interação com o computador. "Ctrl + Alt + T"

ver todas opções do comando "comando + --help"

Manual do comando "man + comando"

Oque o comando faz "whatis + comando"

pesquisar por comando apropriado "apropos + comando"

Acesse o diretório raiz "cd /"

diretório principal "cd ~"

lista os diretórios "ls"

mostra outras opções "ls + Tab"

permite navegar entre as linhas "comando + Tab"

entrar na pasta "cd + nome da pasta"

limpa o terminal "clear"

cria pasta "mkdir + nome da pasta"

mostra o local atual "comantdo + ."

mostra o local a cima "comantdo + .."

mostra o caminho completo "pwd"

Manipulando arquivos e tratando inputs e outputs

criar um arquivo "touch"

faz uma cópia "cp + nome do arquivo + nome da cópia"

mostra oque tem no arquivo "cat + nome do arquivo"

move o arquivo "mv + nome do arquivo + local completo de destino"

apagar um arquivo "rm + nome do arquivo"

apagar um diretório "rmdir + nome da pasta"

cria varios arquivos "touch + nome{1..5}.txt"

apaga varios arquivos "rm + nome{1..5}.txt"

abrir arquivos grandes com paginação "less + nome do arquivo" tecla 'espaço' para avançar e 'q' para sair

pesquisar no arquivo "less + nome do arquivo" depois "/palavra-chave" para avançar entre as palavras "m", 'q' para sair

pesquisar no arquivo "grep + -i + palavra-chave + nome do arquivo"

pesquisar frases no arquivo "grep + -i + 'palavra-chave entre aspas simples' + nome do arquivo"

mostra tudo menos a palavra-chave pesquisada "grep + -iv + 'palavra-chave entre aspas simples' + nome do arquivo"

mostra quantidade de vezes que a palavra-chave repete "grep + -ic + 'palavra-chave entre aspas simples' + nome do arquivo"

mostra as 10 primeiras linhas "head + nome do arquivo"

mostra as 5 primeiras linhas "head + -5 + nome do arquivo"

mostra as 10 ultimas linhas "tail + nome do arquivo"

mostra as 5 ultimas linhas "tail + -5 + nome do arquivo"

mostra quantidade de (linhas, palavras e caracteres) "wc + nome do arquivo"

mostra quantidade de linhas "wc + -l + nome do arquivo"

mostra quantidade de palavras "wc + -w + nome do arquivo"

mostra quantidade de caracteres "wc + -m + nome do arquivo"

mostra quantidade de bytes de um arquivo (um caractere pode ter de 1 a 4 bytes) "wc + -c + nome do arquivo"

Buscando ajuda no terminal

Mostra pasta e oque tem dentro da pasta "ls + *"

Pesquisa ocultando o 1º caractere "ls + ?nome do arquivo"
